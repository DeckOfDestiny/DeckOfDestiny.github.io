# Deck of Destiny Website

This is a static GitHub Pages site for `DeckOfDestiny.github.io`.

## Publish on GitHub Pages

1. Create a GitHub repository named `DeckOfDestiny.github.io`.
2. Upload these files to the root of that repository:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `.nojekyll`
3. Commit and push to the `main` branch.
4. Open:
   - `https://deckofdestiny.github.io/`

## Local Preview

Open `index.html` in a browser, or use any simple local server.

## Files

- `index.html`: site structure and rules/play tabs
- `styles.css`: visual design and responsive layout
- `script.js`: playable prototype logic
